int max(int a, int b) {
    return (a > b)? a : b;
}解释一下这里面符号的含义
以下是对这段代码中符号含义的解释：
int max(int a, int b)：
int：表示这个函数返回一个整数类型的值。
max：是函数名，可以根据具体的需求取不同的名字。这里表示这个函数的目的是找到两个数中的较大值。
(int a, int b)：函数的参数列表，接收两个整数类型的参数 a 和 b。
return (a > b)? a : b;：
return：用于从函数中返回一个值。
(a > b)：这是一个条件表达式，判断参数 a 是否大于参数 b。如果 a 大于 b，这个条件表达式的值为真（非零值）；如果 a 不大于 b，这个条件表达式的值为假（零值）。
? a : b：这是一个三元运算符。如果前面的条件表达式 (a > b) 为真，则整个表达式的值为 a；如果条件表达式为假，则整个表达式的值为 b。所以这个语句的作用就是返回参数 a 和 b 中的较大值。
